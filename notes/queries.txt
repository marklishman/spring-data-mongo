
@Query

  Uses store-specific query language.
  Queries can be externalized to a properties file.


Query Derivation
  
  Query methods use query derivation.
  Spring Data removes 'findBy', 'getBy', 'readBy', 'queryBy' or 'countBy' then parses the rest.


Property expressions

  findByAddressZipCode
  findByAddress_ZipCode
